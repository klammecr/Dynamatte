Dynamatte: Handling Dynamic Camera for Video Matting

Achleshwar Luthra 1 Christopher Klammer 1 Richa Mishra 1

Abstract

Decomposing an input video into a set of layers for different objects in the scene is a long-standing fundamental problem in computer vision. We propose decomposing a video into layers of omnimattes - a set of alpha mattes and RGB values, where each layer represents not only the object but also the effects associated with it (such as shadows, reflections, etc.). Prior work that deals with this problem assumes that the video can be mapped to a static canvas thus limiting their solution to videos with limited camera motion, specifically panning camera motion. In our work, we aim to propose a more generic solution that can deal with unconstrained dynamic camera motion. To solve this problem, we propose a new loss function that can account for time-varying background and we also experiment with learnable homographies between consecutive frames as homography estimates can be incorrect when there is a lot of depth variation in the scene. In this report, we show our results on challenging videos, talk about failure cases and potential future directions.
